## Project Overview

A nationwide directory website for **Hauling & Removal Services**, allowing consumers to search by city, state, or zip code. Initial focus on two subcategories:
- **Estate Cleanout Services**
- **Junk Removal**

## Project Status

This project is in the initial setup phase. No framework or stack has been chosen yet. Before writing code, confirm the tech stack with the user.

## Project tech stack

Build this application with Next.js and Supabase as the database to store everything

## Requirements Reference

See `# Requirements of a Directory Website.md` for the full product requirements.

## Key Product Requirements

- Nationwide coverage — search by city, state, and zip code
- Clean, smooth UX/UI
- Directory listings with subcategory filtering (estate cleanout, junk removal)
- Full-stack web application
